## Open the Demo Application

Navigate to `https://botdemo.sales-demo.f5demos.com`. This is a Juice Shop e-commerce application with product listings, user accounts, and a checkout flow.

{/* ![Juice Shop home page](/csd/images/demo-app-home.png) */}

## Review Pages with Form Inputs

Browse pages that collect sensitive data:

<Steps>
1. **Login page** — username and password fields

   {/* ![Login form](/csd/images/demo-app-login.png) */}

2. **Registration page** — email, password, security question
3. **Payment / checkout** — credit card number, CVV, billing address
4. **Customer feedback** — personal information and free-text fields
</Steps>

These are the pages CSD monitors for unauthorized script activity.

## Inspect Loaded Scripts (Optional)

<Steps>
1. Press **F12** to open Chrome DevTools
2. Open the **Sources** tab
3. Expand the script tree to see all loaded JavaScript files
4. Note scripts loaded from third-party domains
</Steps>

Multiple external scripts load on each page. Some dynamically load additional scripts at runtime. CSD tracks all of these.

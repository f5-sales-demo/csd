Esta página proporciona una biblioteca de scripts de simulación de ataques organizados por categoría. Cada script es una función autoejecutable (IIFE) diseñada para pegar en la Consola de DevTools del navegador en la aplicación de demostración. Utilice estos scripts para ejercitar capacidades específicas de detección de CSD durante demostraciones a clientes.

<Aside type="caution">
Estos scripts son **solo para validación previa a producción autorizada**. Todos los destinos de exfiltración son puntos finales públicos seguros (`www.httpbin.org`, `jsonplaceholder.typicode.com`). Todos los scripts inyectados cargan bibliotecas benignas desde CDN públicas.
</Aside>

## Reglas de Seguridad

Todos los scripts en esta biblioteca siguen estas convenciones:

- **Formato IIFE** — autoejecutable, pegar y ejecutar en la Consola de DevTools
- **Destinos de exfil seguros** — `www.httpbin.org`, `jsonplaceholder.typicode.com`
- **Fuentes de scripts seguras** — `cdn.jsdelivr.net`, `esm.sh`, `unpkg.com`, `ga.jspm.io` (cargando bibliotecas benignas)
- **`mode: 'no-cors'`** en todas las llamadas fetch
- **Declaraciones `var`** para compatibilidad con consola
- **Salida de consola** con prefijo `[CSD Demo]` y etiqueta de categoría
- **Sin cargas útiles maliciosas reales**

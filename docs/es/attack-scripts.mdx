Esta página proporciona una biblioteca de scripts de simulación de ataques organizados por categoría. Cada script es una función autoejecutada (IIFE) diseñada para pegarse en la Consola de DevTools del navegador en la aplicación de demostración. Utilice estos scripts para ejercitar capacidades de detección específicas de CSD durante demostraciones con clientes.

<Aside type="caution">
Estos scripts son **solo para validación autorizada en entornos de preproducción**. Todos los destinos de exfiltración son endpoints públicos seguros (`www.httpbin.org`, `jsonplaceholder.typicode.com`). Todos los scripts inyectados cargan bibliotecas benignas desde CDNs públicos.
</Aside>

## Reglas de Seguridad

Todos los scripts de esta biblioteca siguen estas convenciones:

- **Formato IIFE** — autoejecutables, se pegan y ejecutan en la Consola de DevTools
- **Destinos de exfiltración seguros** — `www.httpbin.org`, `jsonplaceholder.typicode.com`
- **Fuentes de scripts seguras** — `cdn.jsdelivr.net`, `esm.sh`, `unpkg.com`, `ga.jspm.io` (cargando bibliotecas benignas)
- **`mode: 'no-cors'`** en todas las llamadas fetch
- **Declaraciones `var`** para compatibilidad con la consola
- **Salida en consola** prefijada con `[CSD Demo]` y una etiqueta de categoría
- **Sin cargas maliciosas reales**

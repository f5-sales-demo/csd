## Abrir la Aplicación de Demostración

Navegue a `https://botdemo.sales-demo.f5demos.com`. Esta es una aplicación de comercio electrónico Juice Shop con listados de productos, cuentas de usuario y un flujo de pago.

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Página principal de Juice Shop" />

## Revisar Páginas con Campos de Formulario

La página de inicio de sesión es la página principal utilizada en esta demostración:

<Steps>
1. Navegue a la **Página de inicio de sesión** (`/#/login`)

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Formulario de inicio de sesión" />

   CSD clasifica automáticamente los campos de correo electrónico y contraseña como sensibles, lo que hace que esta página sea ideal para demostrar la detección de formjacking. La sección [Activar Detección](/csd/trigger-detection/) utiliza esta página.
</Steps>

La aplicación tiene páginas adicionales con campos de formulario que CSD monitorea. Cada página a continuación recopila datos que un script de formjacking podría exfiltrar:

| Página | Ruta | Campos Sensibles | Categoría de Datos |
| --- | --- | --- | --- |
| Registro | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | correo electrónico, contraseña, pregunta y respuesta de seguridad | Credenciales, recuperación de cuenta |
| Olvidé mi Contraseña | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | correo electrónico, respuesta de seguridad, nueva contraseña | Credenciales |
| Pago | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | nombre del titular, número de tarjeta, fecha de vencimiento | Financiero |
| Dirección | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | nombre, número de teléfono móvil, dirección, ciudad, código postal, país | PII |
| Contacto | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | autor, comentario | PII |
| Queja | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | correo electrónico, mensaje, carga de archivo | PII |
| Cambiar Contraseña | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | contraseña actual, nueva contraseña | Credenciales |

CSD monitorea todas las páginas donde la inyección de JavaScript está habilitada, no solo las listadas anteriormente.

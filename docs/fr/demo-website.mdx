## Ouvrir l'application de démonstration

Accédez à `https://botdemo.sales-demo.f5demos.com`. Il s'agit d'une application e-commerce Juice Shop avec des listes de produits, des comptes utilisateur et un flux de paiement.

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Page d'accueil Juice Shop" />

## Examiner les pages avec des entrées de formulaire

La page de connexion est la page principale utilisée dans cette démonstration :

<Steps>
1. Accédez à la **page de connexion** (`/#/login`)

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Formulaire de connexion" />

   CSD classe automatiquement les champs email et mot de passe comme sensibles, ce qui rend cette page idéale pour démontrer la détection de formjacking. La section [Trigger Detection](/csd/trigger-detection/) utilise cette page.
</Steps>

L'application dispose de pages supplémentaires avec des entrées de formulaire que CSD surveille. Chaque page ci-dessous collecte des données qu'un script de formjacking pourrait exfiltrer :

| Page | Route | Champs sensibles | Catégorie de données |
| --- | --- | --- | --- |
| Inscription | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | email, mot de passe, question de sécurité et réponse | Identifiants, récupération de compte |
| Mot de passe oublié | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | email, réponse de sécurité, nouveau mot de passe | Identifiants |
| Paiement | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | nom du titulaire, numéro de carte, date d'expiration | Financier |
| Adresse | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | nom, numéro de téléphone, adresse, ville, code postal, pays | Informations personnelles |
| Contact | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | auteur, commentaire | Informations personnelles |
| Plainte | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | email, message, chargement de fichier | Informations personnelles |
| Changer le mot de passe | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | mot de passe actuel, nouveau mot de passe | Identifiants |

CSD surveille toutes les pages où l'injection JavaScript est activée, pas seulement celles énumérées ci-dessus.

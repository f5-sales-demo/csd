## Ouvrir l'application de démonstration

Accédez à `https://botdemo.sales-demo.f5demos.com`. Il s'agit d'une application e-commerce Juice Shop avec des listes de produits, des comptes utilisateurs et un processus de paiement.

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Page d'accueil Juice Shop" />

## Examiner les pages avec des champs de formulaire

La page de connexion est la page principale utilisée dans cette démonstration :

<Steps>
1. Accédez à la **Page de connexion** (`/#/login`)

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Formulaire de connexion" />

   CSD classe automatiquement les champs d'e-mail et de mot de passe comme sensibles, ce qui fait de cette page un choix idéal pour démontrer la détection du formjacking. La section [Déclenchement de la détection](/csd/trigger-detection/) utilise cette page.
</Steps>

L'application dispose de pages supplémentaires avec des champs de formulaire que CSD surveille. Chaque page ci-dessous collecte des données qu'un script de formjacking pourrait exfiltrer :

| Page | Route | Champs sensibles | Catégorie de données |
| --- | --- | --- | --- |
| Inscription | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | e-mail, mot de passe, question de sécurité &amp; réponse | Identifiants, récupération de compte |
| Mot de passe oublié | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | e-mail, réponse de sécurité, nouveau mot de passe | Identifiants |
| Paiement | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | nom du titulaire de la carte, numéro de carte, date d'expiration | Financier |
| Adresse | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | nom, numéro de mobile, adresse, ville, code postal, pays | DPI |
| Contact | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | auteur, commentaire | DPI |
| Réclamation | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | e-mail, message, téléchargement de fichier | DPI |
| Changement de mot de passe | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | mot de passe actuel, nouveau mot de passe | Identifiants |

CSD surveille toutes les pages où l'injection JavaScript est activée — pas uniquement celles listées ci-dessus.

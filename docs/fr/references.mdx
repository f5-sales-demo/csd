## PCI DSS

- [PCI DSS v4.0 Standard](https://www.pcisecuritystandards.org/document_library/) — Bibliothèque complète des documents du standard
- **Exigence 6.4.3** — Gérer tous les scripts des pages de paiement : maintenir un inventaire, fournir une autorisation et une justification écrites, vérifier l'intégrité
- **Exigence 11.6.1** — Déployer des mécanismes de détection de falsification sur les pages de paiement pour alerter sur les modifications non autorisées des en-têtes HTTP et du contenu des pages

## Recherche sur les menaces

- [OWASP Client-Side Security Risks](https://owasp.org/www-project-web-application-security-testing/) — Guide de test de sécurité des applications web OWASP
- [Magecart Threat Research](https://www.riskiq.com/what-is-magecart/) — Aperçu des groupes Magecart et des campagnes d'écrémage numérique
- **Violation de données British Airways (2018)** — Le groupe Magecart 6 a injecté un skimmer dans la page de paiement de BA, compromettant 380 000 transactions
- **Violation de données Ticketmaster (2018)** — Attaque de la chaîne d'approvisionnement via un script chatbot Inbenta compromis qui a écrémé les données de cartes de paiement

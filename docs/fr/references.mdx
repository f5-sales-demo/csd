## PCI DSS

- [Standard PCI DSS v4.0](https://www.pcisecuritystandards.org/document_library/) — Bibliothèque complète des documents standards
- **Exigence 6.4.3** — Gérer tous les scripts de page de paiement : maintenir un inventaire, fournir une autorisation et une justification écrites, vérifier l'intégrité
- **Exigence 11.6.1** — Déployer des mécanismes de détection de falsification sur les pages de paiement pour alerter sur les modifications non autorisées des en-têtes HTTP et du contenu de la page

## Recherche sur les menaces

- [Risques de sécurité côté client OWASP](https://owasp.org/www-project-web-application-security-testing/) — Guide de test de sécurité des applications web OWASP
- [Recherche sur les menaces Magecart](https://www.riskiq.com/what-is-magecart/) — Aperçu des groupes Magecart et des campagnes de capture de données
- **Violation British Airways (2018)** — Le groupe Magecart 6 a injecté un programme d'extraction de données dans la page de paiement BA, compromettant 380 000 transactions
- **Violation Ticketmaster (2018)** — Attaque de la chaîne d'approvisionnement via un script de chatbot Inbenta compromis qui a capturé les données de carte de paiement

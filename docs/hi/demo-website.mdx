## Demo Application खोलें

`https://botdemo.sales-demo.f5demos.com` पर नेविगेट करें। यह एक Juice Shop ई-कॉमर्स एप्लिकेशन है जिसमें प्रोडक्ट लिस्टिंग, यूजर अकाउंट और चेकआउट फ्लो हैं।

## फॉर्म इनपुट वाले पेजों की समीक्षा करें

लॉगिन पेज इस डेमो में प्राथमिक रूप से उपयोग किया जाने वाला पेज है:

<Steps>
1. **Login page** (`/#/login`) पर नेविगेट करें

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Login form" />

   CSD ईमेल और पासवर्ड फील्ड को स्वचालित रूप से संवेदनशील के रूप में वर्गीकृत करता है, जिससे यह पेज formjacking डिटेक्शन का प्रदर्शन करने के लिए आदर्श है। [Trigger Detection](/csd/trigger-detection/) सेक्शन इस पेज का उपयोग करता है।
</Steps>

एप्लिकेशन के पास फॉर्म इनपुट वाले अतिरिक्त पेज हैं जिन्हें CSD निरीक्षण करता है। नीचे दिए गए प्रत्येक पेज ऐसे डेटा एकत्र करते हैं जिन्हें एक formjacking स्क्रिप्ट एक्सफिल्ट्रेट कर सकती है:

| Page | Route | Sensitive Fields | Data Category |
| --- | --- | --- | --- |
| Registration | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | email, password, security question &amp; answer | Credentials, account recovery |
| Forgot Password | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | email, security answer, new password | Credentials |
| Payment | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | cardholder name, card number, expiry | Financial |
| Address | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | name, mobile number, address, city, zip, country | PII |
| Contact | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | author, comment | PII |
| Complaint | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | email, message, file upload | PII |
| Change Password | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | current password, new password | Credentials |

CSD उन सभी पेजों पर निरीक्षण करता है जहां JavaScript इंजेक्शन सक्षम है — केवल ऊपर सूचीबद्ध पेजों तक सीमित नहीं।

## Apri l'Applicazione Demo

Accedi a `https://botdemo.sales-demo.f5demos.com`. Si tratta di un'applicazione e-commerce Juice Shop con elenchi di prodotti, account utente e un flusso di checkout.

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Home page Juice Shop" />

## Rivedi le Pagine con Input Modulo

La pagina di accesso è la pagina principale utilizzata in questa demo:

<Steps>
1. Accedi alla **pagina di accesso** (`/#/login`)

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Modulo di accesso" />

   CSD classifica automaticamente i campi email e password come sensibili, rendendo questa pagina ideale per dimostrare il rilevamento del formjacking. La sezione [Trigger Detection](/csd/trigger-detection/) utilizza questa pagina.
</Steps>

L'applicazione ha pagine aggiuntive con input modulo che CSD monitora. Ogni pagina sottostante raccoglie dati che uno script formjacking potrebbe efiltrare:

| Pagina | Route | Campi Sensibili | Categoria Dati |
| --- | --- | --- | --- |
| Registrazione | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | email, password, domanda di sicurezza e risposta | Credenziali, recupero account |
| Password Dimenticata | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | email, risposta di sicurezza, nuova password | Credenziali |
| Pagamento | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | nome titolare carta, numero carta, scadenza | Finanziario |
| Indirizzo | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | nome, numero cellulare, indirizzo, città, CAP, paese | PII |
| Contatti | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | autore, commento | PII |
| Reclamo | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | email, messaggio, caricamento file | PII |
| Cambio Password | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | password attuale, nuova password | Credenziali |

CSD monitora tutte le pagine in cui l'iniezione JavaScript è abilitata — non solo quelle elencate sopra.

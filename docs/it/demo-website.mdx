## Aprire l'Applicazione Demo

Navigare a `https://botdemo.sales-demo.f5demos.com`. Si tratta di un'applicazione e-commerce Juice Shop con elenchi di prodotti, account utente e un flusso di checkout.

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Juice Shop home page" />

## Esaminare le Pagine con Campi di Input

La pagina di login è la pagina principale utilizzata in questa demo:

<Steps>
1. Navigare alla **Pagina di Login** (`/#/login`)

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Login form" />

   CSD classifica automaticamente i campi email e password come sensibili, rendendo questa pagina ideale per dimostrare il rilevamento del formjacking. La sezione [Attivare il Rilevamento](/csd/trigger-detection/) utilizza questa pagina.
</Steps>

L'applicazione dispone di pagine aggiuntive con campi di input monitorati da CSD. Ciascuna delle pagine elencate di seguito raccoglie dati che uno script di formjacking potrebbe esfiltrare:

| Pagina | Percorso | Campi Sensibili | Categoria Dati |
| --- | --- | --- | --- |
| Registrazione | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | email, password, domanda e risposta di sicurezza | Credenziali, recupero account |
| Password Dimenticata | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | email, risposta di sicurezza, nuova password | Credenziali |
| Pagamento | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | nome del titolare, numero carta, scadenza | Finanziari |
| Indirizzo | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | nome, numero di cellulare, indirizzo, città, CAP, paese | PII |
| Contatto | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | autore, commento | PII |
| Reclamo | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | email, messaggio, caricamento file | PII |
| Cambio Password | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | password attuale, nuova password | Credenziali |

CSD monitora tutte le pagine in cui è abilitata l'iniezione JavaScript — non solo quelle elencate sopra.

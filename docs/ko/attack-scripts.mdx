이 페이지는 카테고리별로 정리된 공격 시뮬레이션 스크립트 라이브러리를 제공합니다. 각 스크립트는 데모 애플리케이션의 브라우저 DevTools 콘솔에 붙여넣기하도록 설계된 자체 실행 함수(IIFE)입니다. 이 스크립트들을 사용하여 고객 데모 시 특정 CSD 탐지 기능을 실행할 수 있습니다.

<Aside type="caution">
이 스크립트들은 **승인된 사전 프로덕션 검증 전용**입니다. 모든 데이터 유출 대상은 안전한 공개 엔드포인트(`www.httpbin.org`, `jsonplaceholder.typicode.com`)입니다. 모든 주입된 스크립트는 공개 CDN에서 무해한 라이브러리를 로드합니다.
</Aside>

## 안전 규칙

이 라이브러리의 모든 스크립트는 다음 규칙을 따릅니다:

- **IIFE 형식** — 자체 실행, DevTools 콘솔에 붙여넣기 후 바로 실행
- **안전한 유출 대상** — `www.httpbin.org`, `jsonplaceholder.typicode.com`
- **안전한 스크립트 소스** — `cdn.jsdelivr.net`, `esm.sh`, `unpkg.com`, `ga.jspm.io` (무해한 라이브러리 로드)
- 모든 fetch 호출에 **`mode: 'no-cors'`** 적용
- 콘솔 호환성을 위한 **`var` 선언** 사용
- `[CSD Demo]` 및 카테고리 태그가 접두사로 붙는 **콘솔 출력**
- **실제 악성 페이로드 없음**

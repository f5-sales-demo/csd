## Demo Application 열기

`https://botdemo.sales-demo.f5demos.com`으로 이동하세요. 이는 상품 목록, 사용자 계정, 결제 흐름이 있는 Juice Shop 전자상거래 애플리케이션입니다.

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Juice Shop home page" />

## 양식 입력이 있는 페이지 검토

로그인 페이지는 이 데모에서 사용되는 주요 페이지입니다:

<Steps>
1. **Login page**(`/#/login`)로 이동

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Login form" />

   CSD는 이메일 및 비밀번호 필드를 자동으로 민감한 데이터로 분류하므로 이 페이지는 formjacking 탐지를 시연하기에 이상적입니다. [Trigger Detection](/csd/trigger-detection/) 섹션에서 이 페이지를 사용합니다.
</Steps>

애플리케이션에는 CSD가 모니터링하는 양식 입력이 있는 추가 페이지가 있습니다. 아래의 각 페이지는 formjacking 스크립트가 유출할 수 있는 데이터를 수집합니다:

| 페이지 | 경로 | 민감한 필드 | 데이터 카테고리 |
| --- | --- | --- | --- |
| Registration | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | email, password, security question &amp; answer | 자격증명, 계정 복구 |
| Forgot Password | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | email, security answer, new password | 자격증명 |
| Payment | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | cardholder name, card number, expiry | 금융 |
| Address | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | name, mobile number, address, city, zip, country | 개인정보 |
| Contact | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | author, comment | 개인정보 |
| Complaint | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | email, message, file upload | 개인정보 |
| Change Password | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | current password, new password | 자격증명 |

CSD는 위에 나열된 페이지뿐만 아니라 JavaScript 주입이 활성화된 모든 페이지를 모니터링합니다.

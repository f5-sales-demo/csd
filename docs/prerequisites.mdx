## Requirements

| Item | Detail |
| --- | --- |
| **F5 XC Console** | Account with CSD permissions under Common Services |
| **Demo application** | `https://botdemo.sales-demo.f5demos.com` loads in browser |
| **Browser** | Chrome recommended (DevTools used in later steps) |
| **Email** | Access to the configured CSD alert receiver (optional) |

<Aside type="caution">
If the Client-Side Defense tile is not visible under Common Services, contact your tenant administrator.
</Aside>

Confirm the demo application loads and you can reach the login page before proceeding.

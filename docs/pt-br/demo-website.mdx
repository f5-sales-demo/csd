## Abrir a Aplicação de Demonstração

Acesse `https://botdemo.sales-demo.f5demos.com`. Esta é uma aplicação de e-commerce Juice Shop com listagens de produtos, contas de usuário e um fluxo de checkout.

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Juice Shop home page" />

## Revisar Páginas com Entradas de Formulário

A página de login é a principal utilizada nesta demonstração:

<Steps>
1. Navegue até a **página de Login** (`/#/login`)

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="Login form" />

   O CSD classifica automaticamente os campos de e-mail e senha como sensíveis, tornando esta página ideal para demonstrar a detecção de formjacking. A seção [Acionar Detecção](/csd/pt-br/trigger-detection/) utiliza esta página.
</Steps>

A aplicação possui páginas adicionais com entradas de formulário que o CSD monitora. Cada página abaixo coleta dados que um script de formjacking poderia exfiltrar:

| Página | Rota | Campos Sensíveis | Categoria de Dados |
| --- | --- | --- | --- |
| Registro | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | e-mail, senha, pergunta e resposta de segurança | Credenciais, recuperação de conta |
| Esqueci a Senha | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | e-mail, resposta de segurança, nova senha | Credenciais |
| Pagamento | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | nome do titular, número do cartão, validade | Financeiro |
| Endereço | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | nome, número de celular, endereço, cidade, CEP, país | PII |
| Contato | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | autor, comentário | PII |
| Reclamação | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | e-mail, mensagem, upload de arquivo | PII |
| Alterar Senha | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | senha atual, nova senha | Credenciais |

O CSD monitora todas as páginas onde a injeção de JavaScript está habilitada — não apenas as listadas acima.

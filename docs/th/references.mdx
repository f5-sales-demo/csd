## PCI DSS

- [มาตรฐาน PCI DSS v4.0](https://www.pcisecuritystandards.org/document_library/) — คลังเอกสารมาตรฐานฉบับสมบูรณ์
- **ข้อกำหนด 6.4.3** — จัดการสคริปต์ทั้งหมดในหน้าชำระเงิน: รักษารายการสินค้าคงคลัง ให้การอนุมัติและเหตุผลเป็นลายลักษณ์อักษร และตรวจสอบความสมบูรณ์
- **ข้อกำหนด 11.6.1** — ติดตั้งกลไกตรวจจับการดัดแปลงในหน้าชำระเงินเพื่อแจ้งเตือนเมื่อมีการแก้ไข HTTP headers และเนื้อหาหน้าเว็บโดยไม่ได้รับอนุญาต

## การวิจัยภัยคุกคาม

- [ความเสี่ยงด้านความปลอดภัยฝั่งไคลเอนต์ของ OWASP](https://owasp.org/www-project-web-application-security-testing/) — คู่มือการทดสอบความปลอดภัยเว็บแอปพลิเคชันของ OWASP
- [การวิจัยภัยคุกคาม Magecart](https://www.riskiq.com/what-is-magecart/) — ภาพรวมของกลุ่ม Magecart และการรณรงค์ขโมยข้อมูลดิจิทัล
- **การละเมิดของ British Airways (2018)** — Magecart Group 6 ฝัง skimmer เข้าไปในหน้าชำระเงินของ BA ส่งผลกระทบต่อธุรกรรม 380,000 รายการ
- **การละเมิดของ Ticketmaster (2018)** — การโจมตีห่วงโซ่อุปทานผ่านสคริปต์ Inbenta chatbot ที่ถูกบุกรุก ซึ่งขโมยข้อมูลบัตรชำระเงิน

## หมวดหมู่การโจมตีและมาตรฐาน

- [OWASP Clickjacking](https://owasp.org/www-community/attacks/Clickjacking) — คำจำกัดความและการป้องกันการโจมตี UI redressing
- [OWASP Man-in-the-Browser](https://owasp.org/www-community/attacks/Man-in-the-browser_attack) — การโจมตีการดักจับผ่านเบราว์เซอร์
- [OWASP Cross-Site Scripting](https://owasp.org/www-community/attacks/xss/) — อนุกรมวิธานการโจมตีแบบ script injection
- [MITRE ATT&amp;CK T1195 Supply Chain Compromise](https://attack.mitre.org/techniques/T1195/) — กรอบการโจมตีห่วงโซ่อุปทาน
- [MITRE ATT&amp;CK T1185 Man-in-the-Browser](https://attack.mitre.org/techniques/T1185/) — เทคนิคการจัดการเบราว์เซอร์
- [MITRE ATT&amp;CK T1496 Resource Hijacking](https://attack.mitre.org/techniques/T1496/) — การจัดประเภท Cryptojacking
- [MITRE ATT&amp;CK TA0010 Exfiltration](https://attack.mitre.org/tactics/TA0010/) — กลวิธีการขโมยข้อมูล
- [Akamai Web Skimming](https://www.akamai.com/glossary/what-is-web-skimming) — ภาพรวม Digital skimming
- [Sansec Magecart Research](https://sansec.io/what-is-magecart) — กลุ่ม Magecart และการรณรงค์

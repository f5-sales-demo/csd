本页面提供了按类别组织的攻击模拟脚本库。每个脚本都是一个自执行函数（IIFE），设计用于粘贴到演示应用程序的浏览器 DevTools 控制台中。在客户演示期间，使用这些脚本来验证特定的 CSD 检测能力。

<Aside type="caution">
这些脚本**仅用于授权的预生产验证**。所有数据外泄目标均为安全的公共端点（`www.httpbin.org`、`jsonplaceholder.typicode.com`）。所有注入的脚本从公共 CDN 加载良性库。
</Aside>

## 安全规则

本库中的所有脚本遵循以下约定：

- **IIFE 格式** — 自执行，在 DevTools 控制台中粘贴即运行
- **安全的外泄目标** — `www.httpbin.org`、`jsonplaceholder.typicode.com`
- **安全的脚本源** — `cdn.jsdelivr.net`、`esm.sh`、`unpkg.com`、`ga.jspm.io`（加载良性库）
- 所有 fetch 调用使用 **`mode: 'no-cors'`**
- 使用 **`var` 声明**以确保控制台兼容性
- **控制台输出**以 `[CSD Demo]` 和类别标签为前缀
- **无实际恶意载荷**

## PCI DSS

- [PCI DSS v4.0 标准](https://www.pcisecuritystandards.org/document_library/) — 完整标准文档库
- **要求 6.4.3** — 管理所有支付页面脚本：维护清单、提供书面授权和理由说明、验证完整性
- **要求 11.6.1** — 在支付页面上部署篡改检测机制，针对 HTTP 头和页面内容的未授权修改进行告警

## 威胁研究

- [OWASP 客户端安全风险](https://owasp.org/www-project-web-application-security-testing/) — OWASP Web 应用安全测试指南
- [Magecart 威胁研究](https://www.riskiq.com/what-is-magecart/) — Magecart 组织与数字窃取攻击活动概述
- **英国航空数据泄露事件（2018）** — Magecart Group 6 向英航支付页面注入窃取脚本，导致 380,000 笔交易数据泄露
- **Ticketmaster 数据泄露事件（2018）** — 通过被入侵的 Inbenta 聊天机器人脚本发起供应链攻击，窃取支付卡数据

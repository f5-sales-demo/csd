本頁面提供按類別組織的攻擊模擬腳本庫。每個腳本是一個自執行函數（IIFE），設計用於貼入示範應用程式的瀏覽器 DevTools 主控台。在客戶示範期間使用這些腳本來練習特定的 CSD 偵測功能。

<Aside type="caution">
這些腳本僅用於**授權的前期生產驗證**。所有資料外洩目標都是安全的公開端點（`www.httpbin.org`、`jsonplaceholder.typicode.com`）。所有注入的腳本都會從公開 CDN 加載無害的程式庫。
</Aside>

## 安全規則

此庫中的所有腳本遵循這些慣例：

- **IIFE 格式** — 自執行，在 DevTools 主控台中貼上並運行
- **安全的資料外洩目標** — `www.httpbin.org`、`jsonplaceholder.typicode.com`
- **安全的腳本來源** — `cdn.jsdelivr.net`、`esm.sh`、`unpkg.com`、`ga.jspm.io`（加載無害的程式庫）
- **所有 fetch 呼叫中的 `mode: 'no-cors'`**
- **`var` 宣告**用於主控台相容性
- **主控台輸出**前綴為 `[CSD Demo]` 和類別標籤
- **無實際惡意載荷**

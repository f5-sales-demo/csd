## 開啟演示應用程式

瀏覽至 `https://botdemo.sales-demo.f5demos.com`。這是一個具有產品清單、使用者帳戶和結帳流程的 Juice Shop 電子商務應用程式。

<Screenshot light="/images/demo-app-home.png" dark="/images/demo-app-home.png" alt="Juice Shop 首頁" />

## 檢視具有表單輸入的頁面

登入頁面是此演示中使用的主要頁面：

<Steps>
1. 瀏覽至**登入頁面**（`/#/login`）

   <Screenshot light="/images/demo-app-login.png" dark="/images/demo-app-login.png" alt="登入表單" />

   CSD 自動將電子郵件和密碼欄位分類為敏感資料，使此頁面非常適合演示表單劫持偵測。[觸發偵測](/csd/trigger-detection/)部分使用此頁面。
</Steps>

該應用程式有其他具有表單輸入的頁面，CSD 會監控這些頁面。以下每個頁面都收集表單劫持指令碼可能外洩的資料：

| 頁面 | 路由 | 敏感欄位 | 資料類別 |
| --- | --- | --- | --- |
| 註冊 | [`/#/register`](https://botdemo.sales-demo.f5demos.com/#/register) | 電子郵件、密碼、安全問題與回答 | 認證、帳戶復原 |
| 忘記密碼 | [`/#/forgot-password`](https://botdemo.sales-demo.f5demos.com/#/forgot-password) | 電子郵件、安全回答、新密碼 | 認證 |
| 支付 | [`/#/payment/shop`](https://botdemo.sales-demo.f5demos.com/#/payment/shop) | 持卡人姓名、卡號、有效期 | 財務資訊 |
| 地址 | [`/#/address/create`](https://botdemo.sales-demo.f5demos.com/#/address/create) | 姓名、行動電話號碼、地址、城市、郵遞區號、國家 | 個人識別資訊 |
| 聯絡 | [`/#/contact`](https://botdemo.sales-demo.f5demos.com/#/contact) | 作者、留言 | 個人識別資訊 |
| 申訴 | [`/#/complain`](https://botdemo.sales-demo.f5demos.com/#/complain) | 電子郵件、訊息、檔案上傳 | 個人識別資訊 |
| 變更密碼 | [`/#/privacy-security/change-password`](https://botdemo.sales-demo.f5demos.com/#/privacy-security/change-password) | 目前密碼、新密碼 | 認證 |

CSD 監控啟用 JavaScript 注入的所有頁面 — 不僅限於上面列出的頁面。

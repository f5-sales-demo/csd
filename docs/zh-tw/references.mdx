## PCI DSS

- [PCI DSS v4.0 標準](https://www.pcisecuritystandards.org/document_library/) — 完整標準文件庫
- **要求 6.4.3** — 管理所有付款頁面腳本：維護清單、提供書面授權與說明、驗證完整性
- **要求 11.6.1** — 在付款頁面部署防竄改機制，以警示未經授權的 HTTP 標頭與頁面內容修改

## 威脅研究

- [OWASP 用戶端安全風險](https://owasp.org/www-project-web-application-security-testing/) — OWASP 網頁應用程式安全測試指南
- [Magecart 威脅研究](https://www.riskiq.com/what-is-magecart/) — Magecart 組織與數位側錄攻擊活動概覽
- **英國航空資料外洩事件（2018 年）** — Magecart 第 6 組將側錄程式注入英國航空付款頁面，導致 380,000 筆交易遭到入侵
- **Ticketmaster 資料外洩事件（2018 年）** — 透過遭入侵的 Inbenta 聊天機器人腳本發動供應鏈攻擊，側錄付款卡資料
